# 🎮 Site PC Gamer - Arnaud C

Site web pour aide au choix de PC portable gamer avec formulaire de contact et sélection de produits.

## 📁 Structure du projet

```
├── index.html          # Page principale (accueil)
├── admin.html          # Page d'administration (privée)
├── script.js           # JavaScript partagé
└── README.md           # Documentation
```

## 🚀 Installation

### Option 1 : GitHub Pages (Recommandé)

1. **Créer un repository GitHub**
   - Allez sur [github.com](https://github.com)
   - Cliquez sur "New repository"
   - Nommez-le `pc-gamer-site` (ou autre nom)
   - Cochez "Public"
   - Cliquez sur "Create repository"

2. **Uploader les fichiers**
   - Cliquez sur "uploading an existing file"
   - Glissez-déposez les 3 fichiers :
     - `index.html`
     - `admin.html`
     - `script.js`
   - Cliquez sur "Commit changes"

3. **Activer GitHub Pages**
   - Allez dans "Settings" du repository
   - Dans le menu de gauche, cliquez sur "Pages"
   - Source : Sélectionnez "main" branch
   - Cliquez sur "Save"
   - Votre site sera disponible à : `https://votre-username.github.io/pc-gamer-site/`

### Option 2 : Netlify

1. Allez sur [netlify.com](https://www.netlify.com)
2. Créez un compte gratuit
3. Glissez-déposez les 3 fichiers
4. Votre site est en ligne !
